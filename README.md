# 📝 Note Taking App (CLI → Streamlit)

A simple yet powerful Note Taking Application built using Python.
Started as a CLI-based project and later upgraded into a web application using Streamlit.

---

## 🚀 Features

* ➕ Create notes
* 📖 View all notes
* 🔍 Read a specific note
* ✏️ Edit existing notes
* ❌ Delete notes
* 💾 Persistent storage using JSON

---

## 🧠 What I Learned

* File handling in Python (`json`, `os`)
* CRUD operations (Create, Read, Update, Delete)
* Writing modular and reusable functions
* Error handling and input validation
* Transitioning from CLI to Web App
* Building UI using Streamlit

---

## 🛠️ Tech Stack

* Python
* Streamlit
* JSON (for storage)

---

## ▶️ Run Locally

1. Clone the repository:

```
git clone https://github.com/your-username/note-taking-app.git
cd note-taking-app
```

2. Install dependencies:

```
pip install streamlit
```

3. Run the app:

```
streamlit run app.py
```
